Telegram Bot
Connect nclaw to Telegram to interact with your agent from anywhere on your phone. Uses long polling — no public server or domain name required.
1
Open Telegram and search for @BotFather, then start a chat with it.
2
Send the /newbot command to create your bot.
/newbot
3
Choose a display name (e.g. My nclaw) and then a unique username ending in _bot (e.g. my_nclaw_bot).
4
BotFather sends you a bot token. It looks like:
1234567890:ABCdefGHIjklMNOpqrSTUvwxYZ
5
Open the .env file in your nclaw directory and add:
TELEGRAM_BOT_TOKEN=your_token_here
6
Restart nclaw. The bot starts automatically when it finds the token.
node src/cli.js
7
Search for your bot in Telegram and send /start. You're connected!
